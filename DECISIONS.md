# Decisiones Tecnicas - Nova Reforma Barcelona

## 1. Stack Inicial

El proyecto usara HTML, CSS y JavaScript sin frameworks.

Motivo:

- reforzar fundamentos de frontend;
- mantener el proyecto simple y claro;
- mostrar capacidad de construir una web profesional sin depender de librerias;
- preparar una base solida antes de avanzar a proyectos con React y TypeScript.

## 2. Idioma del Proyecto

El sitio, README y decisiones tecnicas estaran en espanol.

Motivo:

- el proyecto esta orientado al mercado espanol;
- simula una web para una empresa local de Barcelona;
- mejora la presentacion para reclutadores y clientes en Espana.

## 3. Flujo de GitHub

El proyecto se trabajara con issues, branches y pull requests.

Reglas:

- issues en ingles;
- PR title/body en espanol;
- commits en ingles;
- ramas por issue;
- merge solo despues de revisar el resultado.

## 4. Estado Actual

Repositorio en fase inicial.

La primera issue prepara la estructura base del proyecto.
